# Neumorphism Side Bar Menu using HTML and CSS

Um projeto simples de **sidebar com estilo neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **menu lateral moderno com efeito neumorphism**, usando sombras suaves para um visual elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Sidebar fixa
- Estilo neumorphism
- Layout simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
